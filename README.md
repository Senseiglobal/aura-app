# aura-app
Supabase app with RLS tests and CI
## AURA App

**AURA is a Supabase-powered web app demo.  
It includes secure database setup (RLS), automated tests, and CI/CD integration.**

---

### Features

- User authentication (sign up, login)
- Database tables for artists, songs, and insights
- Row Level Security (RLS) tested and enforced
- Automated CI tests via GitHub Actions
- Easy setup and deployment instructions
